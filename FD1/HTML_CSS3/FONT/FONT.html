Весь фокус состоит в том, как считывается HEX-код. Все сиволы, которые не являются символами шеснадцатиричной системы исчисления заменяются на нуль, далее весь код дополняется нулями до числа, кратного трём, затем делится на три части, и отбрасывваются аргументы таким образом, чтобы от каждой из трёх частей остались только два перых символа. Из трёх образовавшихся пар символов и cкладывается код получаемого цвета.
b 0 0 c / e 0 0 0/ 0 0 0 0
